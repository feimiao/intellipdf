# IntelliPDF Logo设计规范

## 🎯 品牌定位
智能PDF处理工具 - 专业、现代、可靠

## 🎨 设计概念
结合PDF文档、智能处理、工具转换三个核心概念

## 📐 设计规格

### 1. 核心元素
- **PDF图标**：代表文档处理
- **大脑/齿轮**：代表智能处理  
- **箭头/循环**：代表格式转换
- **工具图标**：代表工具箱

### 2. 颜色方案

#### 主色系
```
科技蓝：#2563EB
智能绿：#10B981  
活力橙：#F59E0B
深灰：#374151
浅灰：#F3F4F6
```

#### 使用比例
- 主色（蓝）：60%
- 辅色（绿）：25%
- 强调色（橙）：10%
- 中性色：5%

### 3. 字体规范

#### 英文字体
- **Logo字体**：Montserrat Bold 或 Inter Bold
- **标语字体**：Inter Regular
- **代码字体**：Fira Code 或 JetBrains Mono

#### 中文字体
- **标题**：思源黑体 Bold
- **正文**：思源黑体 Regular

### 4. 尺寸要求

#### 不同场景
```
GitHub头像：      256x256px (正方形)
Open Collective：  400x400px
网站Favicon：     32x32px  
文档Header：      800x200px
社交媒体：        1200x630px
```

#### 格式要求
- 主格式：PNG（透明背景）
- 矢量格式：SVG（可选）
- 打印格式：PDF（高分辨率）

## 🛠️ 设计工具推荐

### 在线工具（免费）
1. **Canva** - https://www.canva.com/
   - 搜索"PDF"和"AI"图标模板
   - 拖拽设计，简单易用

2. **Figma** - https://www.figma.com/
   - 专业设计工具
   - 免费个人版可用

3. **LogoMakr** - https://logomakr.com/
   - 专门Logo设计
   - 大量图标库

4. **Looka** - https://looka.com/
   - AI生成Logo
   - 多种风格选择

### 设计模板

#### 方案A：简洁现代
```
[蓝色PDF图标] + IntelliPDF（深色字体）
+ 下方小字：Intelligent PDF Processing
```

#### 方案B：图标组合  
```
[PDF文件图标]内嵌[大脑图标]
下方：IntelliPDF
```

#### 方案C：抽象符号
```
[三个旋转箭头形成PDF的"P"形状]
右侧：IntelliPDF
```

## 📝 使用注意事项

1. **一致性**：所有平台使用相同Logo
2. **可读性**：小尺寸下依然清晰
3. **适应性**：深色/浅色背景都适用
4. **版权**：确保使用合法图标资源
